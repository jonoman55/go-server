# GO Server

## About the project

This is a simple GO Server with HTTP CRUD routes.

### Commands

Install packages:

`go get`

Start command:

`go run main.go`

### API Routes

| Method | Endpoint | Description |
| ------ | ------ | ------ |
| GET | /posts | Get All Posts |
| POST | /posts | Create Post |
| PUT | /posts/:id | Update Post |
| GET | /posts/:id | Get Post By ID |
| DELETE | /posts/:id | Delete Post |
